# @jood/v-modal

이 패키지는 모달, 컨펌, 액션시트와 같은 동적인 컴포넌트 UI 구성과 \
이런 구성에서 일반적으로 필요한 컴포넌트간의 데이터 통신, 프로그래미틱한 제어의 기반을 제공하는것을 목표로 합니다.

Vue + composition-api + rxjs 를 활용한 Vue 전용 패키지 입니다. \
Angular material 의 DialogRef 에서 영감을 받았습니다.
